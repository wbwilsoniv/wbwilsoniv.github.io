    Remake of the 1978 Milton Bros classic game.

    Description:

    Akin to its handheld predecessor, Simon is a memory game where the user is shown a random pattern of colors and tones, which they must repeat. The game begins with displaying one of the four colors and playing the accompanying sound. After, it is the user’s turn to click the correct color. If correct, the game progresses to the next round, where an additional color is added and displayed after the first. The game continues adding, a single color to the existing pattern each round and gradually displays the pattern faster.
